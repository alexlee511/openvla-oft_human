# Humanized LIBERO-10 Dataset

This dataset contains humanized demonstrations for the LIBERO-10 task suite,
converted from NPZ joint angle trajectories to RLDS format.

The data was generated by:
1. Recording humanized joint angle trajectories (NPZ format)
2. Replaying them in LIBERO simulation environments to render images and compute EEF states
3. Converting the resulting cleaned HDF5 files to RLDS format
